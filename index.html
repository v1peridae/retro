Very Important Tutorial

Installing

  • Install VirtualBox (or any VM hypervisor) on your host OS.
  • Create a new VM and attach the Windows XP ISO image.
  • During setup, enter this product key: VCFQD-V9FX9-46WVH-K3CD4-4J3JM

Getting a Shared Folder

  1. Inside VirtualBox, open Devices → Shared Folders → Settings…
  2. Add a folder, tick Auto-mount and Make Permanent.
  3. In XP, open My Computer and browse to the new drive letter.
  4. If prompted, run VBoxWindowsAdditions.exe to install Guest Additions.
  5. You can now access the shared drive and run your compiled .exe files.
